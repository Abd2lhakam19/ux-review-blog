🚀 The UX Review - Neobrutalism Blog

Welcome to The UX Review, a bold, unfiltered tech and design blog project. This project is built with a Neobrutalist aesthetic, focusing on raw design, high-contrast visuals, and a seamless responsive experience.


🛠️ Tech Stack

    HTML5: Semantic structure for better SEO and accessibility.

    CSS3: Advanced styling including:

        Flexbox: For precise and dynamic layout distribution.

        Desktop-First Responsive Design: Optimized for large screens with smooth scaling for mobile.

        Custom Animations: Interactive transforms, rotations, and hover effects.

    Google Fonts: Integrated "Exo" font for a modern, industrial look.

✨ Key Features

    Neobrutalist UI: Sharp borders, heavy shadows, and high-energy colors.

    Interactive Hero Section: Features a dynamic rotating image with smooth hover transitions.

    Smart Navigation: A responsive navbar that transforms into a mobile menu for smaller devices.

    Advanced Form States: A newsletter subscription box with custom :focus and :not(:placeholder-shown) logic for a premium feel.

    Clean Codebase: Separated concerns with dedicated files for base styles and media queries.

📱 Responsive Breakpoints

The project ensures a consistent experience across all devices:

    Desktop (>1280px): Full multi-column layout.

    Tablet (768px - 1024px): Optimized spacing and mobile-menu integration.

    Mobile (<640px): Vertical stack layout for maximum readability.

🚀 How to Run

    Clone the repository: git clone <your-repo-link>

    Open index.html in your browser.
